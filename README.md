This is a web platform designed to showcase beautiful, unique handmade products from local artisans, helping them display their craft and connect with a larger audience.

## Features
* **Artisan Profiles:** Dedicated space to display information about local creators.
* **Product Gallery:** Beautiful grid layout showing handmade crafts, pottery, textiles, and jewelry.
* **Search & Filter:** Easy navigation to find products by category or artisan.
* **Responsive Design:** Works perfectly on mobile phones, tablets, and desktops.

## Technologies Used
* HTML5
* CSS3
* JavaScript
* db.jason
